**TicketFlow** is a modern, high-performance SaaS platform bridging the gap between cinema owners and movie lovers. Designed with a sleek **Glassmorphism** aesthetic, it offers a seamless experience for ticket reservations and theater management.

---

## ✨ Key Features

### 🍿 For Viewers:
*   **Interactive Seat Map:** Pick your perfect spot using our high-fidelity hall layouts.
*   **Instant E-Tickets:** Get your QR-code ticket immediately after a secure checkout.
*   **Smart Reminders:** Never miss a show with integrated notification system.

### 🏗️ For Cinema Owners (Tenants):
*   **Visual Hall Designer:** Create custom seat grids for any theater type (VIP, IMAX, Standard) with a few clicks.
*   **Dynamic Scheduling:** Effortlessly manage movie sessions, pricing, and hall availability.
*   **Real-time Analytics:** Track ticket sales, revenue, and audience trends via an intuitive dashboard.

---

## 🛠 Tech Stack

*   **Core:** React 18 + Vite
*   **State Management:** Redux Toolkit
*   **Styling:** Tailwind CSS (Custom Dark Glass Theme)
*   **Forms & Validation:** Formik + Yup
*   **Navigation:** React Router v6
*   **API Layer:** RTK Query (Integration with Spring Boot Backend)
*   **Auth:** Google OAuth2 & JWT-based authentication
